# Atlas Charts

Generated by `generate_atlas.py`. All charts use excess-corrected scores (base-rate subtracted).

## Excess Score Methodology

Raw probe scores are inflated by base rates (e.g., if 30% of positions are delimiters, a random head scores 0.30 on delimiter without specialization). Excess scores subtract the step-0 base rate to reveal genuine specialization. See RESULTS.md Finding 6 for details.

## Charts

### developmental-timeline-excess.png

Stacked area chart showing head type counts at every training step. One panel per run (baseline, comparison, seed2; nl-barrier when available). Shows when different specializations emerge and how the distribution stabilizes.

### entropy-three-way.png

Mean attention entropy over training for all runs on one plot. Demonstrates that the entropy trajectory is seed-independent (baseline and seed2 overlap) but tokenizer-dependent (comparison stays lower).

### p0-sink-emergence-excess.png

P0 sink (dormant) head count over training for all runs. Shows merge barriers reduce dormancy.

### specialization-index-excess.png

Mean specialization index over training for all runs.

### layer-depth-specialization-excess.png

Heatmap of per-layer specialization index over training. One panel per run. Shows which layers specialize first and whether the pattern is tokenizer-dependent.

### polysemanticity-excess.png

Specialist (>0.7) vs generalist (<0.3) head counts over training. One panel per run.

### seed-comparison-excess.png

Bar chart comparing baseline vs seed2 head type distribution at step 20000, with distribution correlation (r=0.794). Demonstrates partial stochasticity of emergence.

### circuit-comparison.png

Position heatmap showing the largest co-specializing circuit for each run. Yellow marks positions shared across runs. Demonstrates circuits form the same type but at different positions across seeds.

## Regenerating

```bash
python generate_atlas.py --use-excess       # excess scores (default and recommended)
python generate_atlas.py --dark             # dark theme
python generate_atlas.py --both-themes      # both themes
```
